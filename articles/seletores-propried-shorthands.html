Seletores, propriedades de texto e shorthands

CSS BÁSICO

Seletores

Os seletores são as identificações utilizadas no CSS interno e externo para que a estilização descrita seja aplicada à um item ou itens específicos. Existem alguns tipos de seletores como:

  • Seletores de tipo: os seletores de tipo são aqueles que descrevem o item a ser estilizado pela própria tag HTML, ou seja, se incluímos span { estilo } em nosso código CSS, todas tags span do documento receberão a estilização descrita.
  • Seletores de classe: os seletores de classe estilizam itens com base em suas classes designadas no documento HTML. Como exemplo, para uma tag que possui o atributo class = "vermelho", se inserirmos .vermelho { estilo } em nosso código CSS, todas as tags com a classe vermelho serão estilizadas conforme descrito. Este seletor é muito utilizado para reuso de código, em que blocos de código a serem estilizados da mesma forma recebem classes iguais para que o código de estilização seja um só.
  • Seletores de id: os seletores de id estilizam itens com base em sua id designada no documento HTML. Dessa forma, se temos uma tag com o atributo id = "item", caso tenhamos o código #item { estilo } em nosso documento CSS, a tag com o id item será estilizada conforme descrito. Este seletor é utilizado para estilizações específicas em um item, visto que ids, ao contrário de classes, não podem ser reutilizados em diferentes tags.
  • Seletor universal: o seletor universal, representado pelo símbolo * estiliza o código como um todo. Este seletor é muito utilizado para realizar o "reset" da estilização padrão do navegador, para que se tenha maior controle dos espaçamentos entre itens a serem aplicados, visto que muitas tags possuem espaçamentos padrão definidos pelo navegador. Também é utilizado para definir propriedades padrão que se repetirão ao longo do documento, como fonte, cor de fundo ou cor de texto.
  • Seletor de aninhamento: o seletor de aninhamento, representado pelo símbolo & faz um link entre seletores pais e seletores filhos dentro do documento. Como exemplo, temos a imagem a seguir, em que tags ul terão seu estilo de lista modificado para nenhum, tags li filhas de tags ul terão sua cor de fonte modificada para cinza e tags span filhas das tags li anteriormente citadas, tão seu peso de fonte modificado para 500.
  • Seletores de atributo: os seletores de atributo estilizam itens com base em valores de atributos das tags HTML, em que a estilização descrita será aplicada caso o valor do atributo possua em parte ou em sua totalidade, o valor descrito no seletor. Como exemplo, temos a tag a, que possui como atributo principal o href. Se gostaríamos de estilizar tags a, com atributo href que possua o sufixo .br podemos adicionar o código a[href$=".br"] à folha de estilo. Diferentes símbolos antecedendo o símbolo de igualdade representam diferentes tipos de busca dentro do valor do atributo. O símbolo * representa "contém em qualquer parte do valor", o símbolo ^ representa "inicia com", entre outras simbologias possíveis.
  • Combinação de seletores: os seletores também podem ser combinados para uma maior especificidade daquilo que se está querendo estilizar. Por exemplo, um código como .item p{ estilo } estiliza todos os parágrafos dentro de tags com a classe "item" ou um código span, h3 { estilo } confere estilo igual à todas as tags span e h3 no documento. A depender do símbolo utilizado entre as tags, podendo ser +, >, ~, entre outros, a combinação a ser realizada é modificada.

Propriedades de texto

O CSS possui várias propriedades para estilização de textos. Algumas destas propriedades estão listadas e descritas a seguir:

  • color: esta propriedade modifica a cor da fonte em CSS. Cores em CSS podem ser descritas por nome em inglês (red, black, white), em códigos hexadecimais (#ffffff, #000000), em códigos RGB (rgb(0,0,0)), em código RGBA - quando se quer adicionar a opacidade da fonte por meio desta propriedade - (rgba(0,0,0,0.5)), entre outras formas.
  • font-family: esta propriedade muda a família da fonte. A fonte padrão do documento HTML é a Times New Roman, mas pode ser modificada para uma outra fonte específica que já exista no sistema (como Arial, Georgia, Courier New); para uma nova família sem especificar a fonte (serif, sans-serif, monospace); ou para fontes externas, como o auxílio de uma tag link no documento HTML ou um @import no documento CSS, para que os documentos consigam acessar a fonte.
  • font-size: esta propriedade modifica o tamanho da fonte, podendo receber valores em px, rem, em e porcentagem (%), entre outros.
  • line-height: esta propriedade modifica o tamanho da linha de texto. Algumas fontes, ao aumentar de tamanho, aumentam ainda mais o tamanho de linha. Para modificar questões de espaçamento entre linhas, esta propriedade pode ser utilizada. Assim como o font-size, pode receber valores em px, rem, em e porcentagem (%), entre outros.
  • font-style: esta propriedade modifica o estilo da fonte, recebendo valores como normal, italic ou oblique. O oblique é utilizado para deixar a fonte em itálico, caso uma versão em itálico desta não exista.
  • font-weight: esta propriedade modifica o peso da fonte no texto. Pode receber valores numéricos que vão, usualmente, de 300 a 800 em incrementos de 100, ou valores como bold, bolder e lighter.
  • text-align: esta propriedade modifica o alinhamento do texto, podendo receber valores como left, right, center, entre outros valores relativos ao elemento pai.
  • text-decoration-line: esta propriedade insere uma decoração de linha no texto, podendo receber valores como overline, underline ou line-through, a depender do posicionamento da linha em referência ao texto.
  • text-decoration-color: esta propriedade modifica a cor da linha definida pela propriedade text-decoration-line.
  • text-decoration-style: esta propriedade modifica o estilo da linha definida pela propriedade text-decoration-line, podendo receber valores além do padrão como dotted, dashed e wavy.
  • text-decoration-thickness: esta propriedade modifica a espessura da linha definida pela propriedade text-decoration-line.
  • word-spacing: esta propriedade indica o tamanho do espaço entre palavras em uma frase.
  • letter-spacing: esta propriedade indica o tamanho do espaço entre letras em uma palavra.

Shorthands

Shorthands são uma forma simplificadas e concisas de escrever várias propriedades em uma só linha. Este artifício é muito utilizado para deixar o código mais objetivo e agrupar propriedades que podem ser facilmente agrupadas. Alguns exemplos são:

  • margin, padding e inset (position): podem ser utilizadas com os sufixos top, bottom, left e right, porém, em uma linha só, pode-se definir um valor para todos (1 valor especificado), um valor para top e bottom e um para left e right (2 valores especificados) ou um valor para cada lado do conteúdo (4 valores especificados). Ou seja, um código margin: 1px 2px 3px 4px retorna uma margem de 1px para cima, 2px para a direita, 3px para baixo e 4px para a esquerda. Para padding e inset (definição dos valores de position), o resultado seria o mesmo. Note que, para 4 valores especificados, a sequência dos lados é em sentido horário, iniciando no topo.
  • font: o shorthand para fonte agrupa as propriedades font-style, font-weight, font-size, line-height e font-family. Dessa forma, com o código font: oblique 600 1.4rem/1.5rem, Verdana, sans-serif podemos modificar estas 5 propriedades em uma linha só. Notar que é obrigatório que font-size e line-height sejam descritas uma após a outra e com o separador /.
  • border: o shorthand para border agrupa as propriedades border-width, border-style e border-color. Dessa forma, podemos utilizar o código border: 1px solid #ffffff para modificar as três propriedades em uma linha.
  • background: o shorthand da propriedade background agrupa quatro propriedades: background-color, background-image, background-repeat e background-position. Dessa forma, com o código background: red url(image.jpg) no-repeat left, podemos aplicar as quatro propriedades em uma linha só.
